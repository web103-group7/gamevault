# User Stories

## User Roles
- Player
- Collector

## User Stories

1. As a player, I want to add games to my library so that I can organize inventories for multiple games in one place.

2. As a player, I want to open a specific game page so that I can view only the items that belong to that game.

3. As a player, I want to create a new inventory item with details like name, type, rarity, quantity, and stats so that I can keep track of the gear I own.

4. As a player, I want to edit an existing item so that I can update its stats, quantity, or notes when my inventory changes.

5. As a player, I want to delete an item so that I can remove gear I no longer want to track.

6. As a collector, I want to filter items by game, type, and rarity so that I can quickly find specific categories of items without scrolling through everything.

7. As a collector, I want to sort items by name, power, or date added so that I can browse my inventory in the way that is most useful to me.

8. As a player, I want to create custom loadouts or collections so that I can group items into builds like “Best RPG Gear” or “PvP Setup.”

9. As a player, I want to add items to a loadout so that I can organize which items belong together in a specific build.

10. As a player, I want to see all items inside a selected loadout so that I can review that build in one place.

11. As a player, I want a starter inventory to be automatically created when I add a new game so that I can begin with example items instead of an empty page.

12. As a player, I want to open an item detail modal on the same page so that I can quickly review or edit an item without navigating away from the inventory screen.

13. As a player, I want the app to validate my item inputs so that invalid data like missing names or negative quantities cannot be saved.

14. As a player, I want to see clear error messages when something goes wrong so that I understand what happened and how to fix it.
